# Gulp Project
Para inicializar proyectos con Gulp

## Módulos
```
gulp-sass
browser-sync
gulp-useref
gulp-uglify
gulp-cssnano
gulp-imagemin
gulp-cache
del
run-sequence
```
## Árbol de archivos
```
|- app/
  |- css/
  |- fonts/
  |- images/ 
  |- index.html
  |- js/ 
  |- scss/
|- dist/
|- gulpfile.js
|- node_modules/
|- package.json
```

## 1.- Instalación
```
npm install
```
## 2.- Ejecutar la aplicación
```
gulp
```
## 3.- Crear archivos finales
```
gulp build
```
